# SISREN – Spatial Intelligence and Structured Reasoning for Embodied Navigation

Website for the **SISREN** workshop at **CoRL 2026**.

The workshop examines where spatial intelligence breaks down in embodied agents and how structured reasoning can address these failures, organized around six open research challenges:

- **C1.** Spatial representation learning
- **C2.** Compositional and long-horizon spatial reasoning
- **C3.** Generalizable cognitive maps
- **C4.** Grounding language to spatial reasoning
- **C5.** Uncertainty-aware navigation
- **C6.** Evaluation methodology

Date: **November 9, 2026**  
Location: **Austin, TX**
